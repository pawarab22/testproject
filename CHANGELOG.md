# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-19

### Added
- Initial release of Pooja's Aura Artistry website
- Portfolio management system with image and video support
- Enquiry form for booking requests
- Feedback/Review system
- Admin panel for managing enquiries and feedback
- Password visibility toggle in admin login
- Auto-refresh portfolio updates (3-second interval)
- LocalStorage-based data storage
- Version management system

### Features
- Responsive design for all devices
- Category filtering in portfolio
- Featured looks section on homepage
- Testimonials section with admin replies
- Contact page with quick enquiry form
